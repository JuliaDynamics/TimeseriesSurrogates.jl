Overview

TimeseriesSurrogates.jl

A Julia package for generating surrogate time series.

I'm new to surrogate testing

Then you might want to check out

  1. The method of surrogate testing

  2. What is a surrogate time series?

  3. Types of surrogate realizations

I'm experienced with surrogate testing

Then you're probably want to check out what Types of surrogate realizations this package provides. You'll find:

  1. Random shuffle surrogates (RS), which are just random permutations of the time series.

  2. Fourier surrogates (FS), in the form of either Random amplitude surrogates or Random phase surrogates.

  3. Amplitude adjusted Fourier transform surrogates. Currently, the Amplitude adjusted Fourier transform (AAFT) and Iterated AAFT (AAFT) methods are implemented.

I want to visualize my surrogate realizations

Then you might save some time by checking out the links below. TimeseriesSurrogates.jl provides some convenient plotting routines that'll make it easy to check if your surrogates are successfully capturing your target null hypothesis:

  1. Autocorrelation / periodogram panels. Check out the Examples to get started.

  2. Animate panels (and export to .gif). This allows you to check properties of an ensemble of surrogate realizations.
